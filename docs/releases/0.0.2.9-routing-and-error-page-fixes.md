## Technical Details

### Routing Changes
```php
// Before: Confusing dual routes
$router->get('/wiki/{slug}/redlink', [$wikiController, 'handleRedLink']);
$router->post('/wiki/{slug}/create', [$wikiController, 'createFromRedLink']);

// After: Single consolidated route
// Note: Red links now redirect directly to /wiki/create with title parameter
```

### Redirect Logic
```php
// PageController::show - Non-existent pages now redirect to:
return new Response(
    302,
    ['Location' => '/wiki/create?title=' . urlencode($slug)],
    ''
);
```

### New Catch-All Routes
```php
// Root dohome page routing
$router->get('/{page}', function($request, $page) {
    // Skip core pages that should not be redirected
    $corePages = ['settings', 'profile', 'dashboard', 'search', 'login', 'register', 'auth', 'skins'];
    
    if (in_array(strtolower($page), $corePages)) {
        return new Response(404, ['Content-Type' => 'text/html'], 'Page not found');
    }
    
    // Redirect to wiki namespace
    return new Response(302, ['Location' => '/wiki/' . urlencode($page)], '');
});

// API endpoint routing
$router->get('/api/{endpoint}', function($request, $endpoint) {
    return new Response(404, ['Content-Type' => 'text/html'], 'API endpoint not found');
});
```
