## 🚀 Next Steps (0.1.0 Planning)

### Phase 1: MediaWiki-Style Structure Implementation (v0.1.0)

#### Highest Priority - Core Structure
1. **Root Directory Structure**: Implement MediaWiki-style root organization
   - `index.php`, `api.php`, `load.php`, `thumb.php`
   - `LocalSettings.php`, `composer.json`, `.htaccess`
   - `INSTALL`, `UPGRADE`, `SECURITY`, `COPYING`

2. **Core Folders Implementation**:
   - `/cache/` - Caching system with Islamic content cache
   - `/docs/` - Documentation with Islamic and developer guides
   - `/extensions/` - Plugin system for Islamic features
   - `/images/` - Media storage with Islamic content organization
   - `/includes/` - Core PHP classes with Islamic core
   - `/languages/` - Internationalization (Laravel-style `resources/lang`)
   - `/maintenance/` - Administrative scripts
   - `/mw-config/` - Installation & configuration (renamed to `iw-config`)
   - `/resources/` - Frontend assets
   - `/skins/` - Themes/templates
   - `/sql/` - Database schema
   - `/tests/` - Testing suite
   - `/vendor/` - Third-party dependencies

3. **Configuration System**: Implement hybrid approach
   - `LocalSettings.php` (main config)
   - `IslamSettings.php` (optional Islamic overrides)
   - Fallback to defaults if neither file exists

4. **API System**: Implement hybrid approach
   - `api.php` (main API)
   - `api-quran.php`, `api-hadith.php` (optional overrides)
   - Maintain backward compatibility

#### High Priority - Islamic Integration
1. **Islamic Core Classes**: Implement within `app/Core/Islamic/`
   - `QuranService.php` - Quran text management
   - `HadithService.php` - Hadith collection management
   - `CalendarService.php` - Hijri calendar functions
   - `ArabicTextProcessor.php` - Arabic text processing

2. **Islamic Extensions**: Create core Islamic extensions
   - `QuranicIntegration/` - Quran verse integration
   - `HadithCitations/` - Hadith reference system
   - `IslamicCalendar/` - Hijri calendar system
   - `ArabicEnhanced/` - Enhanced Arabic support

3. **Database Schema**: Implement Islamic-specific schema
   - `quran/` - Quran verses and translations
   - `hadith/` - Hadith collections
   - `islamic/` - Islamic events, scholars

#### Medium Priority - Documentation
1. **Technical Documentation**: Create root-level docs
   - `INSTALL`, `UPGRADE`, `SECURITY`, `HISTORY`
   - `RELEASE-NOTES`, `FAQ`, `COPYING`, `CREDITS`

2. **Islamic Documentation**: Create `docs/islamic/`
   - Content guidelines, scholarly standards
   - Arabic guidelines, citation formats
   - Moderation policy, feature documentation

3. **Developer Documentation**: Create `docs/developer/`
   - API reference, extension development
