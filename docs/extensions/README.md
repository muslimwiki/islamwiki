# Extensions

This section documents first-party extensions available in IslamWiki.

- [EnhancedMarkdown](/docs/extensions/EnhancedMarkdown.md): Advanced Markdown features, progress bars, Islamic helpers.
- [MarkdownDocsViewer](/docs/extensions/MarkdownDocsViewer.md): Docs viewer, sidebar tree, search, root `.md` files.
- [GitIntegration](/docs/extensions/GitIntegration.md): Git commits, backups, review branches.
