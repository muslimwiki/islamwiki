# IslamWiki Extensions

This directory contains documentation for various extensions that extend the core functionality of IslamWiki.

## Available Extensions

### [DashboardExtension](DashboardExtension.md)
A comprehensive, role-based dashboard system that provides personalized experiences for different user roles including Admin, Scholar, Contributor, and basic Users. Features include smart role detection, responsive design, and comprehensive widget system.

### [QuranImportSystem](QuranImportSystem.md)
System for importing Quran content and translations from external sources, with support for multiple languages and translation providers.

### [EnhancedMarkdown](EnhancedMarkdown.md)
Enhanced Markdown processing with Islamic content support, including Arabic text handling and Islamic symbols.

### [MarkdownDocsViewer](MarkdownDocsViewer.md)
Markdown document viewer with support for Islamic content formatting and navigation.

### [GitIntegration](GitIntegration.md)
Git integration for version control and collaboration on Islamic content and documentation.

## Extension Development

For information on developing extensions for IslamWiki, see the [developer documentation](../developer/README.md).

## Extension Standards

All extensions should follow the [extension development standards](../STANDARDS.md) and be compatible with the core IslamWiki framework.
