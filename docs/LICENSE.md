## 🌐 **Network Use Requirements**

### **AGPL-3.0 Network Provision**
The AGPL-3.0 license includes a specific provision for network use:

> "If you run a modified program on a server and let other users communicate with it there, your server must also allow them to download the source code corresponding to the modified version running there."

### **What This Means**
- **Web Applications**: If you run IslamWiki on a web server, you must provide the source code
- **API Services**: If you provide API access, source code must be available
- **Network Access**: Any network-accessible version must include source code access
- **User Rights**: Users must be able to download and modify the source code
