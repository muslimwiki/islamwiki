## 🎯 **Overview**

Welcome to the IslamWiki Wiki System! This comprehensive guide will help you navigate, create, edit, and manage wiki content. The wiki system provides a collaborative platform for sharing Islamic knowledge, articles, and resources.

## 🚀 **Getting Started**

### **Accessing the Wiki**
- **Main Wiki**: Visit `/wiki` to access the wiki homepage
- **Individual Pages**: Access specific pages via `/wiki/{page-slug}`
- **Categories**: Browse content by category at `/wiki/categories`
- **Search**: Use the search function at `/wiki/search`

### **Your First Visit**
1. **Browse Featured Content**: Start with featured pages on the homepage
2. **Explore Categories**: Navigate through different content categories
3. **Search for Topics**: Use the search function to find specific content
4. **Create an Account**: Register to start contributing content

## 📖 **Browsing Wiki Content**

### **Wiki Homepage (`/wiki`)**
The wiki homepage displays:
- **Featured Pages**: Highlighted content and articles
- **Recent Updates**: Latest additions and modifications
- **Category Navigation**: Quick access to content categories
- **Search Bar**: Find specific content quickly
- **Statistics**: Overview of wiki content and activity

### **Category Pages (`/wiki/categories`)**
Browse content organized by categories:
- **Islamic History**: Historical articles and resources
- **Islamic Sciences**: Scientific contributions and discoveries
- **Islamic Art & Architecture**: Artistic and architectural content
- **Islamic Ethics**: Moral teachings and principles
- **Islamic Literature**: Literary works and poetry

### **Individual Page View (`/wiki/{slug}`)**
Each wiki page displays:
- **Page Title**: Clear, descriptive title
- **Content**: Main page content with formatting
- **Meta Information**: Author, creation date, last modified
- **Category**: Page categorization
- **Tags**: Related topics and keywords
- **Actions**: Edit, history, and management options
- **Related Pages**: Suggestions for similar content

## ✏️ **Creating and Editing Content**

### **Creating a New Page**
1. **Navigate to Create Page**: Visit `/wiki/create`
2. **Fill in Page Details**:
   - **Title**: Clear, descriptive title for your page
   - **Content**: Main page content (supports Markdown and HTML)
   - **Meta Description**: Brief summary for search engines
   - **Category**: Select appropriate content category
   - **Content Type**: Choose from page, article, guide, or reference
   - **Tags**: Add relevant keywords separated by commas
3. **Preview Content**: Use the preview button to check formatting
4. **Save Page**: Click "Create Page" to publish

### **Editing Existing Pages**
1. **Access Edit Mode**: Click "Edit" on any page you have permission to edit
2. **Make Changes**: Modify content, title, or metadata
3. **Add Edit Comment**: Briefly describe your changes
4. **Preview Changes**: Check how your edits will look
5. **Save Changes**: Click "Update Page" to apply changes

### **Content Guidelines**

#### **Writing Style**
- **Clear and Concise**: Write in simple, understandable language
- **Islamic Perspective**: Maintain Islamic values and principles
- **Accurate Information**: Verify facts and cite sources when possible
- **Respectful Tone**: Use respectful language appropriate for Islamic content

#### **Content Structure**
- **Introduction**: Start with a clear overview
- **Main Content**: Organize information logically with headings
- **Conclusion**: Summarize key points
- **References**: Include sources and further reading

#### **Formatting Options**
- **Markdown**: Use `#` for headings, `**bold**`, `*italic*`
- **Lists**: Use `-` for bullet points, `1.` for numbered lists
- **Links**: `[text](url)` for external links
- **Images**: `![alt text](image-url)` for images

- **Featured Categories**: Highlighted categories appear on homepage

### **Using Tags**
Tags provide additional content organization:

## 🔍 **Search and Discovery**

### **Basic Search**
1. **Search Bar**: Use the search bar on any wiki page
2. **Enter Keywords**: Type relevant words or phrases
3. **View Results**: Browse through search results
4. **Refine Search**: Use filters to narrow results

### **Advanced Search**
Access advanced search at `/wiki/search`:
- **Search Type**: Choose general, title, content, or category search
- **Category Filter**: Limit results to specific categories
- **Date Range**: Search within specific time periods
- **Sort Options**: Order results by relevance, date, or popularity

### **Search Tips**
- **Use Specific Terms**: More specific terms yield better results
- **Try Synonyms**: Use different words for the same concept
- **Check Spelling**: Ensure correct spelling for accurate results
- **Use Quotes**: Search for exact phrases with quotation marks

## 📚 **Page History and Revisions**

### **Viewing Page History**
1. **Access History**: Click "History" on any wiki page
2. **Revision List**: View all changes made to the page
3. **Compare Versions**: Select two revisions to compare changes
4. **View Specific Revision**: Click on any revision to view that version

### **Understanding Revisions**
Each revision shows:
- **Revision Number**: Sequential revision identifier
- **Date and Time**: When the change was made
- **Editor**: Who made the changes
- **Edit Comment**: Description of what was changed
- **Change Type**: Major or minor edit indicator

### **Reverting Changes**
1. **Select Revision**: Choose the version you want to restore
2. **Click Revert**: Use the revert button to restore that version
3. **Confirm Action**: Confirm the revert operation
4. **Add Comment**: Explain why you're reverting the changes

## 🔒 **Page Protection and Management**

### **Page Locks**
Pages can be locked to prevent editing:
- **Full Lock**: No editing allowed
- **Edit Lock**: Content editing restricted
- **Move Lock**: Page cannot be moved or renamed
- **Delete Lock**: Page cannot be deleted

### **Lock Information**
Locked pages display:
- **Lock Status**: Current lock type and reason
- **Locked By**: Who applied the lock
- **Lock Date**: When the lock was applied
- **Expiration**: When the lock expires (if applicable)

### **Requesting Unlocks**
To request an unlock:
1. **Contact Administrators**: Reach out to wiki administrators
2. **Explain Reason**: Provide justification for unlocking
3. **Wait for Review**: Administrators will review your request
4. **Follow Guidelines**: Ensure you understand why the page was locked
